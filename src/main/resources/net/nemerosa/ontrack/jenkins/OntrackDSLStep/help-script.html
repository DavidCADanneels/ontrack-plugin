Ontrack DSL script to execute.

Two objects are injected into the script:

  • ontrack - base access to Ontrack.
  • jenkins - allows interaction with Jenkins. Supported methods are:
    • env(name, value) - injects an environment variable.
    • result - read only access to the build result.
    • success - read only check to see if the current build is successfull

The execution will fail if one the following conditions is met:

  • an exception occurs.
  • the script returns something else than nothing, 0, false or an empty string.
